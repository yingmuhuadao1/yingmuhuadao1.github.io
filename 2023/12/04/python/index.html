python

print（打印）
pass（关键字）跳过的意思
input（）从用户获取输入，input（）得到的结果一定是字符串。
.upper(将内容变成大写）
.strip(去掉左右两边的空白符）空白符：空白,\t,\n
.replace（old，new）用于字符串的替换。
.split(用什么切割) 用什么切割，就会损失什么。
“用什么连接”join（）连接表格里的内容。
in可以做条件上的判断，或者在for循环语句里。
.startswith 判断字符串是否以xxx开头
.isdigit（）判断字符串是否以整数开头
len（）字符串的长度

如果想把一个数据转换为另一个数据类型。就用想转换成的数据类型，将它包起来，a = int（a）
print（type（a））显示a的数据类型

注释：# 单行注释
“”“多行注释”“”

常量：在python里默认大写为常量

变量：可以发生改变的量，变量是用来区分不同数据的，可以指向一个内存空间，帮我们储存一些数据。
变量的命名规范:
1.必须是数字或字母或下换线组成，
2.不能是数字开头更不能是纯数字不能用python的关键字
3.不要用中文
4.不要太长
5.要有意义I
6.推荐使用下换线命名或者驼峰命名
7.更不能是纯数字不能用python的关键字

数据类型：区分不同的数据，不同的数据类型有不同的作用
数字：+-*/
文字：展示
字符串：str（*****）
表达方式：’’或 “”或‘’‘ ’‘’或“”“ ”“”
操作：+ 左右两端必须是字符串，表示字符串的连接操作
* 一个字符串只能乘以一个数字，表示字符串重复的次数

while 表达式:
循环体

当表达式为True的时候，程序会一直执行循环体代码，直至表达式为False。

if循环：
1，格式：if 条件判断1:
<执行1>
elif 条件判断2:
<执行2>
elif 条件判断3:
<执行3>
else:
<执行4>
判断条件较多时

2,格式：if 条件:
pass1
else:
pass2
条件较少时

for循环想要计数，必须借助于range

例：range(n)，从零数到n，不包含n
range（m，n）：从m数到n。不包含n
range（m，n，s）：从m数到n，不包含n，每次间隔是s。
语法·：for 变量 range（n）：

字符串：
int（整数）
float（小数，浮点数）
小数·：数据范围是有限的
整数：在某一个特点的区间内是可以表示的很清楚
布尔：条件判断
布尔值主要有2个 ture或false

基础数据类型的转换
在python中所有非零的数字都是ture，零是false
在python中所有非空字符串都是ture，空是false
综上，在python中，表示空的东西都是false

字符串的格式化问题
%s 字符串占位
%d 占位整数

字符串的常规操作
字符串的操作一般不会对原字符串产生影响，一般是返回一个新的字符串
