sqlmap
sqlmap五种独特的sql注入技术
时间，布尔，报错，联合，堆叠

在sqlmap中当url地址后面参数大于一个的时候，需要把url地址用双引号包起来。

sqlmap还可以在文件中判断是否存在注入，使用-r命令，-r *.txt，一般使用于存在cookie注入的时候。

–dbs，查看所有数据库
再将–dbs缩写为-D xxx（代表-D代表缩写，xxx代表指定数据库）
–tables，代表查询表，-D xxx –tables代表查询指定数据库中的表）
-D xxx -T xxx -columns （查询指定数据库，指定表，中的字段）
-D xxx -T xxx -C 字段名 –dump（查询指定的字段值 –dump代表拖库，还可以在dump后面加上-all，代表拖出所有的表）
–users（当前用户拥有读取所有用户的权限时，可以使用该命令，查看所有用户）
–passwords（查看所有用户的密码）
–current -db（查看当前网站使用的数据库）
–current -user（查看当前网站数据库的用户名称）
–level 5（代表探测等级，一共有15级，越高探测的越详细，但是速度会比较慢）
–is-dba （判断是否拥有数据库管理权限）
–roles（列出数据库管理员角色，必须有权限，且是oracle）
–referer url（在等级达到3的时候，会尝试对referer进行注入）
–sql-shell （进入执行sql语句的环境，可以执行任意sql语句）
–os-cmd （执行操作系统的命令）
–os-shell（交互式系统的shell）
–file-read 文件路径（读取文件）
–file-write –file-dest（上传文件到数据库中）
tamper “模块名”（表示使用不同的模块，来进行绕过（4050页）
–identify-waf （识别waf版本）

nmap
namp ip地址（扫描该ip开放的端口）
nmap ip1 ip2（扫描2个ip）
nmap ip1-ip2（扫描2者之间的ip，所开放的端口）
nmap 192.168.2.0/24（扫描这个网段）
nmap -iL 文件地址（文件里存放需要扫描的ip）
nmap 192.168.2.0/24 -exclude 需要排除的ip地址（表示排除需要排除的ip地址）
nmap ip -p 22,23,80（扫描指定的端口）
nmap –traceroute ip（对目标进行路由跟踪）
namp -sP 192.168.2.0/24（扫描该c段的在线情况）
nmap -O ip（对目标进行系统识别）
nmap -sV ip （探测目标开放端口对应服务版本）
nmap -sF -T4 ip（探测目标防火墙开放状态）
nmap -O -Pn/-p0(不使用ping的方式，而且假定所有主机系统都是活动的，可以穿透防火墙，也可以避免被防火墙发现）
dnslog
dnslog的作用：用于解决无回显的问题
dnslog，会随机生成一个域名，如果有请求的话，会在dnslog上面回显。用于xml漏洞等

msf
几大模块
auxiliaries（辅助模块）
exploit（漏洞利用模块）
payload（攻击载荷模块）
post（后渗透模块）
encoders（编码工具模块）

渗透攻击步骤
扫描目标机系统，寻找可用漏洞
选择并配置一个漏洞利用模块
选择并配置一个攻击载荷模块
选择一个编码模块，来绕过杀毒软件查杀。（比较鸡肋，因为msf开源）
渗透攻击

在msf中，可以输入nmap来显示nmap提供的扫描选项列表

后渗透攻击
1，进程迁移：有时候获得的shell是比较脆弱的。这个时候我们需要，将shell绑定在一个稳定的进程里面。
2，输入ps查看进程。输入getpid查看shell进程。
3，migrate 进程号，将shell切换到稳定的进程号

信息收集
sysinfo（查看目标机的系统信息）
run post/windows/gather/checkvm（ 查看目标机是否在虚拟机上。）
idletime（查看目标机的最近运行时间）
route（查看目标机完整地网络设置）
background（将当前会话放入后台）
run post/windows/manage/killav (关闭目标机的杀毒软件）
run post/windows/manage/enable_rdp (启动远程桌面协议，俗称3389）
run post/windows/manage/autoroute(查看目标机的本地子网情况）
route add 添加路由·
route print 查看路由
run post/windows/gather/enum_logged_on_users(列举当前有多少用户）
run post/windows/gather/enum_applications (列举安装在目标上的应用程序）
run windows/gather/credentials/windows_autologin(抓取自动登录密码）
load espia （加载插件espia，屏幕截图）
screengrab（抓取目标机的屏幕截图）
